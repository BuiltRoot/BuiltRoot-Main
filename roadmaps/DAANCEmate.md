# DAANCEmate Roadmap

## Current Phase
- Define topic taxonomy (pharmacology, monitoring, emergencies, anesthesia basics, regulations)
- UX wireframes for modules, quizzes, and dashboards
- Content import plan + item-writing guidelines

## Next Milestones
1. Build question database schema + import tool
2. Develop quiz engine (timed/untimed, review mode, explanations)
3. Performance analytics (weak areas, spaced repetition)
4. Study plan generator (calendar-based)
5. Launch pilot cohort + collect feedback

## Long-Term Goals
- CE tracking by state/role
- Mobile app wrapper
- Team licenses for offices
- Expand to additional dental CE and certification prep courses
